база недвижимости

Раздел имущество (база недвижимости), отображает единую базу недвижимости, сортировка по ЖК. При наличии нескольких офисов продаж есть распределение недвижимости по офисам и менеджерам.

• Выгрузка объявлений: в Авито, ВТБ, Я Недвижимость и т.д (любые площадки).

• Отправка КП по объекту в 1 клик: реквизиты менеджера подставляются.

• Cоздание сделок по недвижимости: руками договора не пишут, все создается автоматом договора, акты, приложения. любые формы + проверка ответственным.

• Отправка документов на регистрацию онлайн: выпуск ЦП.

• Расчет/ ведение собственной рассрочки. + ЛК + приложение. (переделывается в ведение АРЕНДЫ, процесс автоматизирован).

• Ведение отчета бетона на стройке.

• Расчет налогов.

• Аукцион квартир: закрытый, открытый, сайт + приложение.

• Документы: Модуль для создания любых документов. Создаем шаблон, заполняем переменными и указываем, где документ используется.

• Домофон: экономия на слаботочке.

• Камеры: показываем камеры в приложении.

• Расчетный центр: расчет, отправка квитанций на -email, оплата в приложении.

• Голосование / Опрос жителей: удобный способ взаимодействия с жителями.

• Объекты управления: единый модуль управления всеми ЖК, от (рекламы до домофона).

• События: новости на сайт, в приложение для.

• Приложение для телевизоров/ планшетов: cтавятся в ЖК и крутят ролики, управление в CRM.

• Терминал в ЖК. Программа для информационных стоек. Позволяет жителям оплатить ЖКУ, создать заявку, позвонить в аварийную службу и т.д  + платные услуги.
